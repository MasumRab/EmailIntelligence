Each task follows this format:
- **Task ID** (e.g., 008)
- **Task Title**
- **Status** (pending/in-progress/done/blocked/deferred)
- **Priority** (high/medium/low)
- **Initiative** grouping
- **Sequence** number
- **Purpose** statement
- **Success Criteria** (checklist)
- **Subtasks** (numbered 008.1, 008.2, etc.)

  - Purpose statement

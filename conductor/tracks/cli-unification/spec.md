# Specification: CLI Unification\n\n## Overview\nConsolidate all scattered Python scripts and Shell tools into a single, modular CLI framework based on SOLID principles and the Factory/Registry pattern.\n\n## Core Requirements\n- **Unification**: Single entry point via `dev.py`.\n- **Security**: All commands must use `SecurityValidator` for path and input sanitization.\n- **Modularity**: Domain-based grouping (git, task, analysis, infra, automation).\n- **Async**: Native support for asynchronous command execution.\n- **Compatibility**: Maintain functional parity with existing legacy tools.
